What is this project?


The site was created as part of the course Yandex-Workshop: Python back-end developer

Basic goals:
  • Understand how the Visual Studio Code IDE works
  • Improve understanding of OOP principles
  • Familiarize yourself with the Django framework
  • Get an idea about the frontend: HTML and CSS
  • Make the first project for your portfolio
  • Pass everything on time and not fly out of the course

  • Python 3.7
  • Django 2.2.16
  • VS Code
  • Bootstrap
  • Божья помощь

Embedded capabilities according to the customer's specification:

  • Pages: home page, community page, author profile with posts, post card
  • Creating and editing posts
  • Creating comments on a post
  • Working with communities (creation through the admin panel and its optional selection in the post)
  • Ability to subscribe to authors (subscription feed with a switch to the main page)
  • Ability to add images to posts
  • Customized admin panel
  • Working with users: registering new ones, logging in / out, changing passwords and recovering passwords (emulator)

Implemented in addition to the wishes of the customer from the YaP:

  • Страницы: навигация по группам и авторам с авторами-лидерами по постам и комментариям
  • Личный кабинет с полями за рамками стандартной модели (аватары, доп.инфо об авторе)
  • Пользовательские и стандартные аватары
  • Удаление постов
  • Удаление комментариев
  • Система лайков понравившимся постам
  • Отражение информации о лайках и комментариях в лентах постов с рабочими ссылками
  • Возможность поиска в постах по ключевым словам
  • Система тегов для навигации по постам в виде разноцветных кнопок
  • Немного красоты: кнопки действий вместо ссылок, картинки к кнопкам, посты в виде карточек, оптимизация для различных устройств
